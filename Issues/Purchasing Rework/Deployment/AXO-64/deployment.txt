--------------------------------------------------------------------------------------

3. Upload BBC_DOMAINS_BBC-SQL-D01_BBCMAXDEV_dbo_20140918105522.zip package

Deploy Work Flow Changes
--------------------------------------------------------------------------------------
1. Login to Maximo as an admin user.
2. Go To the Migration Manager Application.
3. Upload BBC_WF_BBC-SQL-D01_BBCMAXDEV_dbo_20140918132857.zip package
4. Deploy BBC_WF_BBC-SQL-D01_BBCMAXDEV_dbo_20140918132857.zip package
5. Select the latest revision of the SUBINVWF workflow.
6. Enable the SUBINVWF.
7. Open the latest revision of TOPICWF workflow.
8. Enable the TOPICWF workflow.
9. Activate the TOPICWF workflow.
10. From the select action menu click  "Resynchronize an Active Workflow Process".
11. Select the latest revision of the POAPPR workflow.
12. Enable the POAPPR.
13. Open the latest revision of MAINPOWF workflow.
14. Enable the MAINPOWF workflow.
15. Activate the MAINPOWF workflow.
16. From the select action menu click  "Resynchronize an Active Workflow Process".
